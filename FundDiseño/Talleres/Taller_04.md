# LISTA DE PATENTES:

1. **UV light monitoring system for a UV decontamination apparatus**
   -Resumen: Esta patente describe un sistema de monitoreo de luz ultravioleta (UV) aplicado a dispositivos de desinfección. El sistema es capaz de estimar con precisión la irradiancia UV que recibe una superficie, incluso cuando esta se encuentra en diferentes ángulos respecto a la fuente de luz UV.

3. **Sistema, dispositivo y métodos para el seguimiento de la exposición a la luz**
    (Light exposure tracking system, device, and methods)
   -Resumen: La patente describe un sistema portátil (wearable) que mide la exposición a la radiación UV en tiempo real. El dispositivo recopila datos y los envía a un sistema externo que calcula el índice UV y la dosis acumulada diaria, ayudando a prevenir riesgos para la salud
